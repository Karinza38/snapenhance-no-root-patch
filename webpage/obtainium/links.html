Snapchat and SnapEnhance Config

Instructions

  1. Make sure Obtainium is installed. You can click the Obtainium button at the very bottom of this page to be taken to the Obtainium GitHub.
  2. Tap on the SnapEnhance button below.
  3. Hit "Continue" when Obtainium opens with a popup. Then return to this page.
  4. Tap on the SnapChat button below.
  5. Hit "Continue" when Obtainium opens with a popup.
  6. Now you can either tap the Install button on the apps one at a time or long press each of the apps to select them and hit the Install button at the bottom of the screen.
    • If this is your first time using Obtainium then you will need to grant app install permissions when it asks for them.
  7. Once both apps are installed you can open either of them and it should prompt you to Generate Mappings. Do this.
  8. Now that the apps are set up you can use them just fine and customize your SnapEnhance settings to your liking.
